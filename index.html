Démineur VERSUS

Règles du jeu

  1. Votre objectif est de trouver et d'éviter toutes les mines sur la grille.
  2. Pour voir ce qu'il se trouve sous une case, il suffit de cliquer dessus.
  3. Un clic droit permet de marquer une case potentiellement minée. Cependant, vous êtes limité. Vous avez autant de marqueurs qu'il y a de mines.
  4. Si une case avec une mine est révélée, vous perdez la partie.
  5. Si vous révéler une case sans mine, le nombre exact de mines autour de cette case sera indiquée.
  6. Les difficultés prédéfinies sont en 10x10.

Des bonus sont égalements présents si vous souhaitez les activer. Ils sont déblocables en découvrant des cases non minées.

  • "Reveal" : Permet de révéler une case aléatoirement. Ce bonus évitera toujours les mines. 3 charges maximums.
  • "Retry" :Permet, lors d'un clique sur une mine, de continuer à jouer. Attention toutefois, ce bonus n'est utilisable qu'une fois par partie.
